```
wget --no-check-certificate -O shadowsocks-all.sh https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks-all.sh 

chmod +x shadowsocks-all.sh 

./shadowsocks-all.sh 2>&1 | tee shadowsocks-all.log
```

  第一条命令的第一个字母意思就是下载一个文件，这个文件的地址就是：https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks-all.sh由于是https，所以需要加个“no-check-certificate”这个参数来保证不会出现错误，然后下载下来的文件因为不知道名字，所以需要添加“-O”来重命名为“shadowsocks-all.sh”

  第二条命令的意思就是为shadowsocks-all.sh文件添加一个可执行权限，linux不同于windows，必须要有权限才可以运行，如果你有个文件但没有运行权限，你只能查看但不能运行。

  第三个命令的意思是将错误日记照常输出，然后保存到shadowsocks-all.log文件中，如果出错了的话还可以知道哪里出了错误

  接着就等待出现以下的字，我们输入3接着回车。

```
Which Shadowsocks server youd select: 
1.Shadowsocks-Python 
2.ShadowsocksR 
3.Shadowsocks-Go 
4.Shadowsocks-libev 
Please enter a number (default 1):
```

  接着我们输入密码和端口号，密码建议复杂一点，端口号随便输入，不过要记住而且避免一些比较容易猜出来的端口号，例如什么8888啊88啊6666这些。最后回车就可以开始等待自动配置完成了。
